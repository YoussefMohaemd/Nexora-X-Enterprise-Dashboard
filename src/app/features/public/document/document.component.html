Getting Started

NexoraX is designed to be the foundation of your next enterprise application. It comes pre-configured with Angular 21, Tailwind CSS 4, and a complete design system.

npx create-nexorax-app my-dashboard

Follow the on-screen instructions to set up your project environment.

Installation

1. Install Dependencies

Run the following command in your terminal to install all required packages.

npm install

2. Start Development Server

Launch the dashboard in development mode with HMR enabled.

npm run start

Usage

Signals System

NexoraX leverages Angular Signals for reactive state management throughout the dashboard.
